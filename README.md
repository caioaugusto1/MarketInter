### Installation

Requires Visual Studio - 2017 - to run
Requires MySQL Workbench 

Change <connectionstring in file `${AppSettings}` after clone git repository

Run Migrations command to execute database.

```sh
$ enable-migrations
$ add-migration
$ update-database
```
